# Cómo unir múltiples PDFs 

Este código permite unir todos los archivos PDF de una carpeta en un solo documento.

## Requisitos
- Python
- pypdf

Instala la librería con:
pip install pypdf

## Uso
1. Coloca todos tus PDFs en una carpeta
2. Cambia la ruta en el código
3. Ejecuta el archivo

## Resultado
Se generará un archivo llamado:
libro_completo.pdf
